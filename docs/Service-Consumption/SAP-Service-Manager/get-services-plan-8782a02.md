# get services/plan



<a name="loio8782a02763404a679b21038e33c097f9__section_xcr_2nt_pkb"/>

## Overview



***btp get services/plan*** 

Get details about a specific service plan.



<a name="loio8782a02763404a679b21038e33c097f9__section_fp5_f4t_pkb"/>

## Usage

`btp get services/plan [parameters]`



<a name="loio8782a02763404a679b21038e33c097f9__section_hdy_lpt_pkb"/>

## Parameters


<table>
<tr>
<th valign="top" colspan="2">

Required



</th>
</tr>
<tr>
<td valign="top">

`ID`



</td>
<td valign="top">

The ID of the service binding to view.



</td>
</tr>
<tr>
<td valign="top">

`-sa`, `--subaccount`



</td>
<td valign="top">

The ID of the subaccount associated with the service plan for which to get details. You can omit this parameter if you have already set the context to the subaccount using `btp target --subaccount ID`.



</td>
</tr>
</table>
